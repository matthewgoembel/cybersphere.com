Projects

Details about various projects I've worked on will be listed here.

Project 1: Cybersecurity Game

Developed an interactive cybersecurity learning game that introduces beginners to key cybersecurity concepts. The game includes challenges like password cracking and network security simulation, designed to teach players how to protect systems from cyberattacks.

Project 2: Temperature Stabilization System

Implemented a temperature stabilization system using TCP sockets for inter-process communication. The system stabilizes the temperature by communicating with external processes and recalculating temperature based on incoming data.

Project 3: Stack and Queue Implementation

Created a linked list implementation of stack and queue data structures in C. Collaborated with teammates to implement and test the structures across different languages (C, C++, Python).

Project 4: OS161 System Calls

Developed custom system calls for OS161, including a system call to add three integers and a printchar system call. Enhanced error handling and debugging techniques while working within the OS161 environment.

Project 5: Database Management System

Designed and implemented relational algebra queries for a custom database management system, working with tables for employees, departments, artists, albums, and customers. Focused on optimization and efficient query processing.
